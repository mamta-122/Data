# Swiggy Sales Analysis Dashboard (Excel)
Interactive Excel dashboard to analyze Swiggy sales, orders, ratings, food types, and city/state-wise performance.

Project Objective

To analyze Swiggy sales data using Excel

To understand monthly, weekly, and daily sales trends

To compare Veg and Non-Veg food sales

To identify top-performing cities and states

To support business decision-making using data insights.

Tools & Techniques Used

Microsoft Excel

Data Cleaning

Pivot Tables

Pivot Charts

Excel Formulas

Slicers for interactivity

Key KPIs

Total Sales

Average Rating

Total Orders

Average Order Value

Rating Count
